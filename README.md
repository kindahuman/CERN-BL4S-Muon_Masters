Hey! This is the GitHub repository for our teams submission to the 2026 Cern Beamline for Schools Competition. For our project we analyzed muon tomography for varying astroid cores. This repository witholds all information regarding the simulations which were done in g4beamline. The files included are the .in files(file type for g4beamline) for the 4 materials simulated, as well as the phython script I created for analyzing the data and comparing our results to highland predictions. All information in the python script is changed for each material. If you have any questions, email ryanberger2009@icloud.com. Feel free to email me if you are interested in the proposal as well! Have a great day!
